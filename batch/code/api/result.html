【分館】9/20（水）小学生向けひまわりおはなし会

下新倉分館では、毎月第3水曜日に小学生向けの「ひまわりおはなし会」を行っています。
申し込みは不要です。ぜひ遊びに来てください。

　日時：9月20日（水）16時00分～16時30分

　場所：下新倉児童館（図書館下新倉分館と同じ建物内1階）

　対象：小学生　※幼児も参加可能です　

↑ 7月19日に図書館下新倉分館で開催したひまわりおはなし会の様子

次回10月18日(水)は、図書館下新倉分館キッズスペースで行います。

【本館】（市内中学生対象）ビブリオバトルのバトラー（発表者）を募集します

（市内中学生対象）ビブリオバトルのバトラー（発表者）を募集します

8月17日（木）更新　申込期間を8月31日（木）17時まで延長しました。

9月24日(日)に本館で開催されるビブリオバトルのバトラー（発表者）を募集します。

日時：令和5年9月24日(日)14時から15時30分まで

場所：和光市図書館本館会議室

バトラー（発表者）：市内中学生５名

申込期間と申込方法：8月2日（水）9時30分から8月31日（木）17時までメールにて（先着順）

申込に必要な事項：氏名（かな）、学校と学年、紹介したい本（和光市図書館に所蔵がある本に限ります）

※定員に達し次第、申込みを締め切ります。

※観覧者は当日先着20名までです（事前申込み不要）
